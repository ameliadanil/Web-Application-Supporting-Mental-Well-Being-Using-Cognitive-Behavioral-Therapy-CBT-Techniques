# 🧠 Mental Health Web Application (CBT-Based)

A web application designed to support mental well-being using Cognitive Behavioral Therapy (CBT) techniques.
The app helps users identify negative thought patterns and work on improving their mental state through structured exercises.

## 🚀 Features

* Interactive CBT-based exercises
* User input and reflection tracking
* Simple and intuitive interface
* Focus on mental well-being and self-improvement

## 🧩 Tech Stack

* React (Create React App)
* JavaScript
* HTML & CSS

## 👩‍💻 My Role

I designed and implemented the application as part of my engineering thesis project, focusing on:

* application logic
* user interaction
* frontend structure
